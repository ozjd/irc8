# irc8

This is an expiremental IRC server. It should not be used in production.

# Running

The [run.bat](https://github.com/ozjd/irc8/blob/master/run.bat) file should be used to start the server.<br>

* Windows (Command Prompt)
    ```run [args]```
* Window (PowerShell)
    ```.\run [args]```
* Linux (BASH)
    ```./run.bat [args]```
* MacOS (BASH)
    ```./run.bat [args]```
